FC25 Site Plan

Site Name

FC25: The Future Coders Class of 2025

This name represents a student-led coding community for the 2025 graduating class, sharing tips, projects, and inspiration for future developers.

Optional domain: fc25.dev

Site Purpose

The FC25 site is a hub for aspiring web developers in the class of 2025. It provides learning resources, student project showcases, tips, and a form to join the community.

Scenarios

  • How can I join the FC25 developer community?
  • Where can I find JavaScript projects made by other students?

Color Schema

Chosen colors and usage:

  • Red (#dc2626) – Used for headings, buttons, and accents.
  • Light Gray (#f3f4f6) – Used for backgrounds and sections.

Typography

  • Segoe UI – Used for body text and headings.
  • Courier New – Used in code blocks or project snippets.
